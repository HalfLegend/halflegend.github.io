主要功能：
	1. tab功能扩展—命令补全：各种命令都会有好多参数，比如 rm -rf 很多参数不容易记忆，在zsh下可以输入 rm -  之后，按几下tab，可以把所有的参数及说明展示出来，然后通过tab选择相应的参数。同样，用cd进入文件夹时，多按几次tab，可以直接选择而不用输入了。
	2. 文件解压： 文件解压可以直接运行压缩文件，自动解压：   $  abc.tar.gz
	3.	智能跳转：z命令，后面加进入的路径一部分，它会在历史记录中找此目录。比如有一个/opt/vertica。输入z verti，回车就能进入。输入完z verti后，也可以按tab自动补完，看一下是不是想要进入的目录，中间可以有空格，如z ver bin。（不止这一个命令，还有a, s, d, f等命令，原理类似，详见https://github.com/clvv/fasd）
	4.	ssh快速连接：s1, s2, s3 命令可以直接连接FineBI1 2 3 服务器，而不用再输入 ssh root@192.168.5.100了
	5.	l 命令，替代ls，列出详细信息
	6.	gn => grep net:  查看端口占用，如 gn 8080，可以查看8080端口有无占用
	7.	gp => grep process: 查看进程运行： 如gn vertica，可以查看vertica有无运行
	8.	kill 命令可以自动完成，比如 kill -9 vertica 然后按tab，可以自动把其id填入
	9.	错误提示：如果输入命令后发现其变红了，说明无此命令。


其它小功能：
	1.	使用&后台执行时，请注意会直接显示其pid，方便以后监控与停止。无须再用其它命令查找其pid了。
	2.	进入文件夹，可以不输入cd，比如直接输入 ABC 就会进入ABC文件夹，同样直接输入..就可以返回上一级。 
	3.	有代码提示等插件，默认是关闭的，有需要可以自行打开。
	4.	支持git等插件，进入受git控制的目录后，会展示分支及状态，且git命令有很强大的提示。如果想支持其它功能，可以自行添加插件。
	5.	man是彩色的。
	6.	主题可换，可以换一种显示风格，官方所有主题：https://github.com/robbyrussell/oh-my-zsh/wiki/themes
	7.	修改配置在家目录下，配置：vi .zshrc   主题和插件安装目录：.oh-my-zsh。(所以像环境变量等全局的配置，尽量往全局的文件里写，而不要放在.bashrc或者.zshrc里)
